The Full-on West Hollywood Home Experience

  • ClientIndipendent
  • BriefWeb Design + Code
  • TechnologyHTML + CSS
  • Website9444sierramarplace.com

This website was one of the first responsive websites I made for a client who wanted to promote their West Hollywood house for rent.

At the time responsive web design was an emerging web technology and I wanted to jump straight in but back then not all web browsers could handle all of the new responsive technology. In order for the website to display well across all browsers I chose to build the site using a responsive front-end framework called Foundation. Foundation framework helped give me the essential building blocks, such a navigation menu, a popup gallery and form. It also provided simple to understand code that allowed me to place all the elements onto a grid system.

Since then I have learned much more about how to code responsive websites and now I either use more modern and up to date frontend code (CSS), or a lighter weight user interface framework to build display grid systems and user interface elements.

You can view more of my work that I have done by visiting the portfolio section of my website.
